📊 Superstore Data Analysis — Tableau Dashboard
A comprehensive visual analytics project exploring sales, profit, shipping behaviour, and category‑level performance using the Sample Superstore dataset.
⭐ Overview
This project analyses the popular Sample Superstore dataset to uncover insights across sales performance, profitability, customer behaviour, and operational efficiency. The Tableau dashboard presents these insights through clear, interactive visualisations designed to support data‑driven decision‑making.
The dashboard includes multiple views such as category‑level performance, city‑level breakdowns, shipping delays, profit margins by ship mode, and time‑series trends.

🧹 Data Preparation
The dataset was cleaned and prepared using SQL/Python before being imported into Tableau.
Key steps included:
Handling missing values and inconsistent categories
Standardising date fields and creating time hierarchies
Engineering fields such as profit margin, shipping delay, and order quantity buckets
Creating category and sub‑category groupings for clearer analysis

📈 Dashboard Components
Your dashboard includes several analytical views:
Category vs City Performance — heat‑map style table showing how product categories perform across cities
Profit Margin by Ship Mode & Region — stacked bar chart comparing profitability across shipping methods
Shipping Delay by Category — horizontal bar chart highlighting operational bottlenecks
Discrete Area Chart for Order Quantity — time‑series view showing order volume trends
Category‑Level Summary — high‑level breakdown of sales, profit, and quantity
These views are combined into a single interactive dashboard for seamless exploration.
🔍 Key Insights
Some of the insights uncovered include:
Certain cities show negative profit margins for specific categories
Standard Class shipping dominates volume but has variable profitability across regions
Furniture often experiences higher shipping delays compared to other categories
Order quantities show seasonal patterns visible in the discrete area chart
Sub‑categories such as Tables and Bookcases contribute disproportionately to losses
